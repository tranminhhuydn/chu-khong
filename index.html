Native File System API chưa được hỗ trợ trong trình duyệt này và Trình soạn thảo văn bản đang chạy ở chế độ kế thừa. Ở chế độ kế thừa, các chỉ báo đã sửa đổi tệp không được hiển thị. Nếu đang chạy Chrome, bạn có thể bật Hệ thống tệp gốc API bằng cách bật # native-file-system-api cờ trong chrome: // flags .
Chữ Không là một trình soạn thảo văn bản đơn giản tương tự như notepad giúp dễ dàng chỉnh sửa các tệp văn bản trên hệ thống tệp cục bộ của bạn. Nó chỉ ra cách để dễ dàng sử dụng các API hệ thống tệp gốc HTML5 mới. Để tìm hiểu thêm về các API hệ thống tệp gốc và cách tạo bản trình diễn này, hãy xem Bài viết về API hệ thống tệp gốc trên Kiến thức cơ bản về web hoặc xem mã nguồn trên GitHub .
